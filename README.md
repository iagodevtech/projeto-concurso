git init

-iniciar novo projeto com git

git add <nome-arquivo>

- add os arquivos que estão prontos para serem commitados

git commit -m "mesagem de commit"

- commit os arquivos no histórico

git log

-mostra o log de alterações

git status

- como está o estado de nossas alterações

git diff

- mostra o que tem de alteração.

git merge

- merge  de ramificações, mescla ramificações(branchs)

git branch

- mostra a branch atual

git branch -b

- cria uma nova brach a partir do histórico da branch  atual.

git checkout <nome da branch>

- muda para essa branch

git remote add <nome>A PRIMEIRA COSTUMA SER ORIGIN e <url> https://github.com/iagodevtech/projeto-concurso

- add um novo repositório remoto

git push <nome><nome da branch>

- manda nossas alterações para o repositório remoto, para cada branch

git pull <nome><nome da branch>

- pega as alterações do repositorio remoto e joga para a maquina

git fetch 

- atualiza o nosso histórico local de acordo com o nosso repositório salvo no remoto, atualização, sicronização do local com o remoto.
